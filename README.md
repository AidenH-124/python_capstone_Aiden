# Name of Game: Realm Of Enemies

# Overview:
Realm of Enemies is a simple RPG game with 3 levels and a final boss at the end.
The player can choose to be a Warrior or a Mage, the player has to fight multiple enemies and fight a dragon at the end of the game.
This game focuses on combat mechanics and progression
# How to run:
To run this program, you need to download the code and run the code in python
# Features:
Choose between two character classes, A warrior, focuses on higher HP but weaker attacks- Mage focuses on lower HP but stronger attacks
Turn-based combat system
Multiple enemies per level
Progressive enemy dificulty scaling
Randomized enemy selection
Healing after wins
Saves results to a text file
# Concepts demonstrated:
Object-Orriented programing
Classes and inheritance
Loops
Randomization
# Libraries used:
Random and Time
# Potential upgrades:
More character classes
Items (Potions, weapons)
Special attacks for different character classes
